Aunque 2021 todavía no acaba, el uso del machine learning y la inteligencia artificial ya marca una tendencia de aumento considerable en comparación con 2020. Las compañías en todo el mundo están aprovechando estos recursos para cumplir objetivos como: Esto es posible gracias a las características del machine learning, que permiten crear aplicaciones de software capaces de hacer predicciones sin estar programadas para ello. Las principales compañías ya han introducido ejemplos de machine learning en plataformas de uso diario como Google, Uber, Amazon y Facebook. Este se ha convertido en un importante diferenciador al momento de ofrecer mejores servicios, ya que la precisión, velocidad y eficiencia supera los métodos manuales de procesamiento de datos. Esto explica las proyecciones para el mercado global del machine learning entre 2019-2025: una tasa compuesta de crecimiento anual del 43.8%.  Ahora bien, en el caso específico de México, la implementación del machine learning es un proyecto con muchos desafíos. Un estudio de everis y la MIT Tech Review encontró que las empresas mexicanas tienen un conocimiento muy superficial de esta tecnología, lo que no les permite aprovecharla.
