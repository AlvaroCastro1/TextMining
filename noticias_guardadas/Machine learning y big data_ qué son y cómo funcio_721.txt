           Machine learning y big data son disciplinas que se complementan precisamente a partir de sus diferencias. Las operaciones logísticas generan en su desarrollo una gran cantidad de datos estructurados y no estructurados en cuyo análisis se utilizan técnicas de big data y machine learning, lo que hace relevante conocer de qué manera se complementan, y cuál es el aporte de big data en logística y de machine learning en ecommerce.  Machine learning y big data, junto al internet de las cosas, minería de datos y otras aplicaciones de inteligencia artificial, son ejemplos de innovación tecnológica, que gracias a la automatización optimizan las operaciones logísticas de las empresas.   Son técnicas de análisis de datos, que evidencian el impacto de la ciencia y la tecnología en los negocios actuales, que con el soporte de software informático introducen a las compañías en la transformación digital.  La cadena de suministro produce una enorme cantidad de datos utilizables para su evaluación,  la combinación de la agilidad de las técnicas del big data y la escalabilidad de la inteligencia artificial en logística, entregan más que nunca valor a las empresas al optimizar los procesos. El big data se ocupa de la procedencia y la naturaleza de las grandes cantidades de información que procesa, el machine learning mediante análisis inteligente se encarga del aprendizaje computacional.    Para responder la pregunta ¿qué es machine learning?, podemos decir que es una forma de inteligencia artificial que permite a las máquinas aprender sin ser expresamente programadas para ello. Llamado también aprendizaje automático, mediante algoritmos prepara a los ordenadores para identificar patrones y realizar predicciones.     ¿Qué es big data? La respuesta es la siguiente: un conjunto de técnicas analíticas que procesan una gran cantidad de datos para detectar patrones ocultos y visibilizar información relevante para la toma de decisiones acertadas y la generación de negocios innovadores. La aplicación de las técnicas de big data en ecommerce proporcionan a este sector herramientas para la optimización de la cadena de suministro, especialmente en las operaciones de distribución.       A pesar de utilizar métodos distintos, el machine learning y big data actúan de manera complementaria. La acción de una potencia es el resultado de la otra: el machine learning utiliza IA para dar a los ordenadores la capacidad de aprendizaje a partir de los datos procesados por big data, para que estos realicen tareas automáticas. El big data es la fuente de obtención y adecuación de datos para el machine learning, detecta patrones y hace disponible la información ante los algoritmos del machine learning.     Entender cómo funciona el machine learning parte de la idea de que las máquinas aprenden de los datos, identifican patrones y toman decisiones con muy poca participación humana. Utilizan en su gestión métodos estadísticos y algoritmos de optimización.
