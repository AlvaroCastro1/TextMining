En esta guía te explico qué es GitHub Copilot, qué precios tiene y cómo funciona. Es posible que ya sepas que esta plataforma, propiedad de Microsoft, se ha convertido en una enorme comunidad de desarrolladores que comparten sus creaciones abriendo su código a todo el mundo. De hecho, hay aplicaciones Android para descargar desde GitHub que son geniales. Por ejemplo, además de las webs para aprender a programar, ahora tenemos ChatGPT, es que es capaz de resolver tus dudas de forma más o menos acertada. Sin ir más lejos, GitHub, una plataforma que hace tiempo cayó en manos de Microsoft, ahora está aprovechando las bondades de la IA generativa para ayudar a los desarrolladores a hacer más en menos tiempo. Esta función es la que se conoce como GitHub Copilot. Pero, ¿qué es exactamente?En pocas palabras, se trata de una función basada en grandes modelos de lenguaje que permite a los desarrolladores simplificar algunas de sus tareas. El concepto que persigue esta herramienta es el mismo que ya conocemos en otros productos que incluyen el apellido Copilot. Por ejemplo, ya te he hablado sobre Microsoft Copilot y su versión Pro. Este asistente, que emplea la tecnología de OpenAI en materia de LLM, se postula como un asistente que puede ayudarte a ejecutar tareas repetitivas, sobre todo si están relacionadas con la escritura de textos. La versión Pro de Microsoft Copilot incluso se integra en aplicaciones tan populares como Excel o Word. En estas, es capaz de analizar texto, generarlo, proponer fórmulas y mucho más.GitHub Copilot es capaz de autocompletar código basándose en el contextoDe forma similar, GitHub Copilot es un asistente, pero específicamente creado para ayudar a los desarrolladores. Según indica Microsoft en la página oficial de este producto, ya son más de 55.000 las empresas que han adoptado este sistema para la creación de aplicaciones. Con todo, es bueno tener claro que la propuesta de Microsoft no es que las aplicaciones se creen a golpe de prompt. Más bien, la idea es que GitHub Copilot actúe como un asistente. Esto es posible gracias a sus numerosas funciones. Vamos a conocer cuáles son.Cómo funciona GitHub CopilotGitHub Copilot funciona gracias a la tecnología OpenAI. Eso significa que se vale de modelos como GPT-4 o GPT-4o para entender las peticiones del usuario, procesarlas y ofrecer una respuesta acorde a lo solicitado. No hay que descartar que Microsoft haya hecho algunos ajustes para que el modelo se comporte de manera específica en entornos de desarrollo. Esto tiene todo el sentido del mundo, pues esta es una solución enfocada a un grupo de usuarios muy concreto, no al público en general, como sucede con ChatGPT.Conocer un poco mejor algunas de las funciones de GitHub Copilot te ayudará a entender mejor hasta qué grado puede ayudar a los creadores de aplicaciones, ya sea que trabajen de forma individual como en equipo.Sugerencias de códigoAsí sugiere GitHub Copilot código a los desarrolladoresGitHub Copilot es capaz de proponer sugerencias a medida que se escribe código. Es una funcionalidad que se parece mucho al autocompletado de Word o la función predictiva en los teclados de los móviles. La idea es que el asistente se anticipe al desarrollador y sugiera los siguientes fragmentos de código. Para lograrlo, tiene en cuenta el contexto y evalúa las intenciones del desarrollador en todo momento. Si la propuesta es adecuada, basta con presionar el tabulador para introducirla en el código. De esta manera, cuando las propuestas acertadas se suceden una tras otra, es posible programar algunas partes de la aplicación a golpe de tabulador.Buscador avanzado en la base de conocimiento de cada proyectoOtra cosa que hacen muy bien los grandes modelos de lenguaje es buscar información. Como detectan cuáles son los términos más importantes en una petición, localizan de manera más precisa la información relacionada. Esto permite a los desarrolladores localizar aquella documentación que necesitan de manera más ágil, lo cual les permite tener más tiempo para leerla.Respuestas en forma de chatOtro punto fuerte de los LLM es la vertiente conversacional. Gracias a eso, GitHub Copilot permite chatear directamente con herramientas de desarrollo especializadas. Además, la compañía planea agregar extensiones y servicios de terceros para que el chatbot sea todavía más útil y se integre mejor con otras herramientas para desarrolladores.Integración con diversos IDEIDE con los que es compatible GitHub CopilotCada desarrollador puede preferir su entorno. Por el momento, es posible utilizar GitHub Copilot en VS Code y Visual Studio, ambos de Microsoft. También en Neovim y en JetBrains IDEs. Adicionalmente, el copiloto de GitHub se ha integrado en su aplicación para móviles con la finalidad de resolver dudas de forma rápida y sencilla. La app de GitHub está disponible para Android y iOS de forma totalmente gratuita.Precios y versiones de GitHub CopilotEn la actualidad, GitHub Copilot se puede contratar en las siguientes modalidades:Versión Individual. 10 dólares por usuario y mes.Versión Business. 19 dólares por usuario y mes.Versión Enterprise. 39 dólares por usuario y mes.Todas estas versiones incluyen las funcionalidades más básicas de chat, sugerencias de código y explicaciones basadas en el contexto, así como la herramienta de localización de errores. De hecho, las suscripciones Individual y Business son idénticas, salvo porque la última incluye diversas opciones para administrar usuarios y configuraciones de roles de acceso. Por su parte, la versión Enterprise es la más completa, incluyendo todas las herramientas, además de acceso a las nuevas características que ahora mismo se mantienen en beta.Finalmente, con respecto a la suscripción a GitHub Copilot debes saber esto:Hay prueba gratuita. Solo disponible para algunos perfiles, como estudiantes o profesores verificados. La plataforma ofrece 30 días gratuitos.Modelos personalizados para empresas.
